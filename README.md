# Elden Ring 100% Tracker

**https://mcnewcp.github.io/elden-ring-100-tracker/**

A single-page web app for tracking 100% completion in Elden Ring. Check off bosses, quests, key items, achievements, and more as you play through the game. Includes a build guide with leveling milestones and gear progression.

Progress syncs across devices via GitHub API, with localStorage as a fallback.
